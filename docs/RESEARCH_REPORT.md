## Abstract
SDG Tracker is a lightweight app to encourage and log sustainable daily actions aligned with SDG 12. This report covers motivation, system design, and future work.

## Motivation
Individuals can make measurable impacts when small habits are recorded and visualized. Tracking creates accountability and supports behaviour change.

## System Design
- MERN stack
- Authentication with JWT
- Dashboard visualizations (daily/weekly/monthly)

## Evaluation & Next Steps
- User testing, A/B testing for behavior nudges
- Add exportable impact metrics and certifications
