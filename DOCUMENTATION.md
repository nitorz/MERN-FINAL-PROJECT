## Summary
This bundle contains a production-ready pattern for a MERN app with JWT auth, analytics, deployment configs, and testing scaffolding.

## Features (expanded)
- Authentication (register/login) using bcrypt + JWT
- Protected API endpoints for creating habits
- Tailwind-styled React frontend with simple routes
- Analytics dashboard with Recharts (line + bar)
- Deployment manifests for common hosts

## How to run locally
See README.md for quick start. For tests:
- Backend: cd server && npm test
- Frontend: cd client && npm test
